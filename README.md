**Process Core Optimizer** is a high-performance system utility built with WPF (Windows Presentation Foundation) designed to give you total control over your CPU resources. Optimize your gaming and professional applications by manually assigning CPU Affinity, managing priorities, and automating performance profiles.

---

## 🌟 Key Features

### 🛠️ Advanced Process Management
- **CPU Affinity Control:** Pin specific applications to high-performance cores and avoid efficiency cores (E-Cores) to eliminate stuttering in games like CS2 or Valorant.
- **System Priorities:** Change process priority levels from "Idle" to "Real-Time" with a single click.
- **Automated Profiles:** Save custom optimization profiles for your favorite games. The app automatically detects when the game starts and applies your saved Affinity and Priority settings in the background.

### 📊 Real-Time Hardware Monitoring
- **Live Stats:** Track **CPU** and **GPU** load and temperatures.
- **Detailed Metrics:** Monitor RAM usage, VRAM, GPU HotSpot temperatures, and Power Draw (TDP).
- **Core-Specific Data:** View individual core clocks and temperatures to identify thermal throttling.

### ⚙️ Usability & UI
- **Modern Dark UI:** A sleek, intuitive interface built with the latest WPF standards.
- **System Tray Integration:** Runs discretely in the background with a fully localized context menu.
- **Bilingual Support:** Full support for both **English** and **Polish** languages.
- **Start with Windows:** Option to launch minimized on system startup for "set and forget" optimization.

## 📥 Download & Installation

The application is distributed as a standalone installer.

1. Go to the [**Releases**](https://github.com/9Erza/ProcessCoreOptimizer/releases) tab.
2. Download the latest installer: `ProcessCoreOptimizer_Setup.exe`.
3. Run the installer and follow the on-screen instructions.
4. **Note:** It is highly recommended to run the application as an **Administrator** to allow the software to modify process priorities and affinities for system-protected applications.
